## Altermative assets Investment |Simulator

**Objective**: 

1. Drive clients investment allocation into alternative assets taking advantage of the market momentum **by showing in simple terms the rate opportunity** along with the advisor speach.
2. Create a tool that was simple but useful to interact with the advisor market momentum speach presenting the main 3 benefits/ features of each type of investment.
3. Deliver a simple, easy and quick tool, easy to use by anyone that also present to the client an idea of the decision impact in the investment amount.

**Data Set**

Based on historic rate data of specific portfolios, fixed income and alternative asset investment.

**Process**

1. Stablish the objective, to interact in a simple way the pros and cons of the investment by choosing between 2 types of assets. The interaction was accomplished by the use of **radio buttons** in excel.
2. The file has a **macro** that allows the level of the graphs to change everytime one type of investment is chosen.
3. The file has **future value formulas** to show when necessary the interest return according to the capital invested.
